This README file explains how to use the MowIt program.

An executable jar file can be found under /bin directory
It should be ran using the following command within a cmd/shell.

  java -jar mowit-1.0.jar [-options]
  
Use option "h" to display all options

  java -jar mowit-1.0.jar -h

When passing a file to the application:
  File lines can contain leading and ending blank caracters
  Empty lines will be ignored
